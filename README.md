##### 게임 동작 방식

- 게임시작 버튼 클릭 시
- 임의의 시간초가 감소하고
- 당근 클릭 시 카운트 감소
- 벌레 클릭 시 패배
- 당근이 남은 채 시간초가 0이 되면 패배
- 임의의 시간초 내에 카운트 0이 되면(당근을 다 뽑으면 🥕) 승리

##### UI/UX 개선점?

- 당근, 벌레 hover 시 크기를 늘림으로써 직관성 개선
- 당근, 벌레 랜덤 생성 시 필드를 정함으로써 UI개선
